# Components for Physical AI & Humanoid Robotics Book

This directory contains all the custom React components for the Physical AI & Humanoid Robotics Book. Each component is organized in its own subdirectory with its own documentation.

## Component Categories

### HardwareSpecs
Components for displaying hardware specifications with tables, comparison views, and cost breakdowns.

### TechnicalDiagrams
Components for displaying technical diagrams, architecture visuals, and system designs.

### CodeExamples
Components for displaying code examples with syntax highlighting, language tabs, and execution contexts.

### BookNavigation
Components for book navigation, progress tracking, and chapter organization.

### ThemeSwitcher
Components for theme management and technical documentation styling.

## Usage

All components are exported through the main index.tsx file and can be imported as:

```tsx
import { HardwareSpecTable, TechDiagram } from './components';
```
